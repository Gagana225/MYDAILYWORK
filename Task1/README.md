# Task 1: Number Guessing Game

## Description
This is a Java console-based Number Guessing Game developed as part of my internship task.

## Objective
To practice Java basics such as loops, conditionals, and user input handling.

## How It Works
- The program generates a random number.
- The user tries to guess the number.
- The program gives hints until the correct number is guessed.

## Technologies Used
- Java
